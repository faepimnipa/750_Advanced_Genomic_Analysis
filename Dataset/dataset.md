# Downloading the dataset 
## This is a personalised data set available under my MWS user name, and my results will be compared to those expected from this data set.
### Download in Ubantu with this command, after replacing the text in <> with my MWS user name.

```bash
wget -R "index.html*" -r -nd -nH -np https://cgr.liv.ac.uk/454/acdarby/LIFE750/Life750_datasets/dataset_<yourMWSusername>/
```
